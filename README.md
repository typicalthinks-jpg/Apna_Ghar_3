# 🎯 Overview

**Apna Ghar** is a modern property portal that connects property owners and tenants directly.

No middlemen, no brokerage — just pure connections! 🏡

### 🌟 Why Apna Ghar?

* ✅ Direct owner-tenant connect
* ✅ Zero brokerage
* ✅ Verified properties
* ✅ Real-time chat
* ✅ Dark mode support
* ✅ Mobile responsive

# 📱 How to Use

## As a Customer

```
1. Sign Up → Select "Customer"
2. Login with credentials
3. Browse properties
4. Search by city / price
5. Save favorites
6. Contact owner
```

## As an Owner

```
1. Sign Up → Select "Owner"
2. Login to dashboard
3. Add new property
4. Upload photos
5. Manage listings
6. View analytics
```
